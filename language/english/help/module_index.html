Random Quote Module Feaures

This module includes the following features:

  • Allows Administrator to create quotes
  • Quotes can be taken offline without deletion
  • Supports using XoopsEditors - selected through preferences
  • Contains plugin support for the Waiting Block module
  • Contains plugin support for the XOOPS Tag module

Random Quote Module Future Enhancements

This module has future plans to support the following features:

  • Category support to allow administrator to put quotes in categories
  • Group support to allow administrator to restrict access to certain categories
  • Notification (by category) when quote is added
  • Ability to allow certain groups to submit quotes from front-side
  • Ability for administrator to monitor (approve/disapprove) online submissions
